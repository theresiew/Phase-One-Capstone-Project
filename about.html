📖 What is Book Explorer?

Book Explorer is a responsive, interactive web application built as part of the Frontend Phase 1 Capstone project. It lets users browse thousands of books, search by title, and save their favorites — all powered by the Open Library API.

✨ Features

🔍

Search Books

Search by title using the Open Library API in real time.

❤️

Save Favorites

Add or remove books from your personal favorites list.

💾

Persistent Storage

Favorites are saved in localStorage and survive page refresh.

📱

Fully Responsive

Works beautifully on mobile, tablet, and desktop.

🛠️ Tech Stack

HTML5 CSS3 Tailwind CSS JavaScript ES6+ Open Library API localStorage GitHub
