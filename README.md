# 📚 Book Explorer

A responsive, interactive, and API-powered book browsing web app built for the Frontend Phase 1 Capstone.

## Features
- 🔍 Search books using Open Library API
- ❤️ Add/remove favorites persisted in localStorage
- 📱 Fully responsive on mobile, tablet, and desktop
- 🗂️ Multi-page: Home, Favorites, About

## Tech Stack
- HTML5, CSS3, Tailwind CSS
- JavaScript ES6+ (modules, async/await)
- Open Library API
- GitHub for version control

## Live Demo
[View Live Site](https://phase-one-capstone-project-pied.vercel.app)

## Pages
- **Home** - Browse and search books fetched from API
- **Favorites** - View and manage saved books
- **About** - Project info and tech stack
